#### Requirements
* Python 
* Django 

#### Usage
~~~python

    python -m django --version
    source env/Script/activate

    pip install django
    pip install djangorestframework
    django-admin startproject example .
    
    ./manage.py migrate
    ./manage.py createsuperuser

~~~

#### Run the Server
~~~python
 
 python ./manage.py runserver
 
 ~~~
